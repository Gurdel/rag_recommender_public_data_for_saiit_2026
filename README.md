# RAG Recommender Public Data for SAIIT-2026

This repository contains public materials prepared as a snapshot for the
SAIIT-2026 conference. The materials are related to a conference submission
on a RAG-based recommender for Ukrainian books.

The repository is intended to make the public data and accompanying
experimental artifacts accessible for review and reference. It includes the
public book dataset, recommender code, and evaluation outputs that
correspond to this snapshot.

The recommender itself remains under active development. We continue to adjust
the system, improve its behavior, and implement new features. Those later
changes will be described in future separate publications.

## Repository Contents

- `books_public_denormalized.csv` - public denormalized book data.
- `RAG/` - main recommender code for the snapshot.
- `e2e testing/` - end-to-end evaluation scenarios and results.
- `nlu testing/` - NLU parser outputs and related evaluation notes.
- `nlu_alternative_parsers_combined.ipynb` - experimental notebook with
  alternative NLU parser approaches.
